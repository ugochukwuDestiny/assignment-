1. Difference Between git diff and git status

  • git diff: Shows actual line-by-line changes not yet staged.
  • git status: Displays file-level changes and staging status, but not file contents.

2. Difference Between git push origin main and git push origin -u main

  • git push origin main: Pushes local main branch to remote origin.
  • git push origin -u main: Same as above, but sets upstream for future git push commands.

3. What is a .env File?

Used to store environment variables like API keys, ports, database connections, etc.

Example:

PORT=3000
DB_URL=mongodb://localhost:27017/mydb
SECRET_KEY=superSecret123

4. What is node_modules?

  • Directory where npm stores installed packages.
  • Auto-generated using npm install.
  • Very large in size, usually ignored with .gitignore.

5. What Does README.md Do?

  • Markdown file that explains what your project does.
  • Includes instructions, usage, installation, etc.
  • Displayed automatically on platforms like GitHub.

Example:

# My Cool Project

A simple Node.js app with environment configs.

## Install
npm install

## Run
npm start
